## Quick Start

<Steps>
  <Step title="Install OpenCode">
    Make sure you have OpenCode 1.0.150 or higher installed
  </Step>
  <Step title="Install Oh My OpenAgent">
    Run the interactive installer:
    ```bash
    bunx oh-my-opencode install
    ```
  </Step>
  <Step title="Start Using Ultrawork">
    Type `ultrawork` or `ulw` in your prompt and watch the magic happen
  </Step>
</Steps>
